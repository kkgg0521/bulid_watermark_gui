# bulid_watermark_gui
Blind&amp;Invisible Watermark （图片盲水印，提取水印无须原图！) 增加图形界面
# how to use
pip install -r requirements.txt
运行main.py
选择 图片文件夹 水印文件夹 出图文件夹 点击开始
# 支持功能
1.批量操作支持多水印多图片
2.支持透明底图片
# 缺陷
处理速度较慢
不支持中文（暂未处理）
